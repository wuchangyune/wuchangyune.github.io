什么是脚手架？

脚手架是工具，是用来辅助工程师进行项目的构建、运行等相关工作，他可以是一个完整的架构，也可以是项目中的一个小架构，这个架构有一个非常显著的特征，复制性强，也称之为可复制模块。

什么是Vue-cli?

vue-cli是vue的脚手架工具，它是帮助前端工程师快速构建vue项目(基础代码工具),构建的基础功能有：项目目录结构，本地调试，代码部署，热加载以及单元测试。

Vue-cli的使用

  1. 全局安装vue-cli工具

    安装指令-npm/cnpm install vue-cli -g;-g 表示全局安装,安装成功之后vue -V 或者vue –version 查看当前vue库的版本
    命令：vue list 查看当前vue的现有的工程模板
    模板的分类：官方模板，简洁版，自定义模板

  2. 创建vue-cli 工程

    通过工程模板进行快速构建项目,vue init webpack projectname

  3. 启动项目

    npm run dev #用来启动开发模式下的应用程序，并不是真正意义上的生产模式(上线模式)

    项目目录

  • 文件说明

    bulid用来配置webpack打包的一些相关设置,config 针对开发环境和上线环境配置.src 项目源代码,index.html是页面内容,mian.js是程序入口

    vue的.vue文件

    一个vue后缀的文件就是一个组件
    一个vue文件标准组成是由三个部分组成
    1.template 表示当前文件的内容模板，组件的ui内容
    2.script 表示当前组件的相关逻辑
    3.style 表示当前组件的样式

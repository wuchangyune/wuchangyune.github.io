2019-02-22

什么是模块？

模块就是具有一点功能的一个组件

Es6的模块？

Nodejs引出的模块概念,规定每一个js文件就是一个模块

Es6种如何使用模块？

通过两个关键字使用模块,通过export和import-导出和导入,
js是不允许外部直接访问的，需要对其进行开发放,开放的目的只有一个(注册)
通过 module.export

export 和 import

export 用于对外输出模块变量(变量，函数，对象)的接口,
import 用于在一个模块种加载另外一个含有export接口的模块

  • 区别
    export 和 default 都可以到处常量，函数，文件，模块等。
    可以在其他js或则模块中使用import的方式导入
